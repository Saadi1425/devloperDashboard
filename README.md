# Dev Dashboard

An interactive personal dashboard built with **HTML, CSS, and JavaScript**.  

### Features
- Live clock displaying current time
- Weather updates (via API)
- Task manager to add and track tasks
- Quote generator with a button to refresh quotes
- GitHub profile integration showing your status
